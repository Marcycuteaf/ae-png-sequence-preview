## Features

- Multiple folders, nested tree browser, search
- Auto-play on select, scrub, FPS control
- One-click add to timeline at playhead
- Auto project bins, custom UI colors
- Loop button and auto Loop

---

## Interface

| Area | Description |
| --- | --- |
| Top | Add folder, clear, theme |
| Left | Search, sequence tree |
| Right | Preview, playback, FPS, Loop, add to timeline |
| Bottom | Status bar |

**Windows:** Shift + click “Add folder” for debug info.

---

## Install

1. Download `.zxp` from [Releases](https://github.com/Marcycuteaf/ae-png-sequence-preview/releases/latest)
2. Install with [ZXP Installer](https://aescripts.com/learn/zxp-installer/)
3. Restart AE → **Window → Extensions → PNG 序列預覽**

---

## Usage

1. Click **Add folder**, pick a root directory
2. Click a sequence in the tree to preview
3. Double-click or press **Add to timeline**

---

## Requirements

After Effects 2019–2025+ · macOS / Windows
