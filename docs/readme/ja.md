## 概要

After Effects 内で多層フォルダーの PNG シーケンスを素早く閲覧し、リアルタイムでプレビュー、ワンクリックで再生ヘッド位置にタイムラインへ追加できます。複数フォルダー、検索、自動分類、UI 配色に対応。

---

## インターフェース

### 上部ツールバー

| 項目 | 説明 |
| --- | --- |
| **➕ フォルダー追加** | 素材のルートフォルダーを選択（複数可）。Windows は Explorer を優先。**Shift + クリック**でデバッグ情報 |
| **🗑** | 読み込み済みフォルダーとシーケンスをすべてクリア |
| **🎨** | 配色パネルの表示／非表示 |
| **ルート表示** | 読み込み済みパスの概要 |

### 配色パネル（🎨）

| 項目 | 説明 |
| --- | --- |
| **メイン / サブ / 背景** | パネル色をカスタマイズ |
| **プリセット** | 組み込み配色をワンクリック適用 |
| **リセット** | デフォルトに戻す |

### 左側 — シーケンスツリー

| 項目 | 説明 |
| --- | --- |
| **🔍 検索** | シーケンス名で絞り込み |
| **件数バッジ** | 見つかった PNG シーケンス数 |
| **⊞ / ⊟** | すべて展開／すべて折りたたみ |
| **ツリー** | `📁` フォルダー、`🎞` シーケンス（枚数表示）。クリックでプレビュー |

### 右側 — プレビューと操作

| 項目 | 説明 |
| --- | --- |
| **プレビュー** | 現在のコマ。未選択時はヒント表示 |
| **名前 / コマ情報** | シーケンス名と `現在 / 総コマ数` |
| **スライダー** | コマ送り |
| **◀ / ▶** | 前／次のシーケンス |
| **▶ 再生** | 再生／停止（スペースキー） |
| **FPS** | プレビュー再生速度 |
| **自動再生** | シーケンス切替時に自動再生 |
| **自動 Loop** | タイムライン追加時に `loopOut("cycle")` を適用 |
| **🔁 Loop** | **選択中のタイムライン層**にループを適用 |
| **⬇ タイムラインへ** | 読み込み＆アクティブコンポの**再生ヘッド**に追加。パック名でビン自動作成 |

### ステータスバー（下部）

読み込み進捗、インポート結果、エラー、デバッグメッセージ（Windows フォルダー選択の失敗理由など）。

## インストール

1. [aescripts ZXP Installer](https://aescripts.com/learn/zxp-installer/) をインストール。
2. [Releases](https://github.com/Marcycuteaf/ae-png-sequence-preview/releases/latest) から OS 用 `.zxp` をダウンロード。
3. インストーラーで導入。
4. AE を再起動 → **Window ▸ Extensions ▸ PNG 序列預覽**。

**手動インストール（macOS）**：CEP 拡張フォルダーへコピーし、AE バージョンに合わせて `PlayerDebugMode` を有効化（2024≈CSXS.11、2025≈CSXS.12）。

---

## 使い方

1. **➕ フォルダー追加** でルートを選択（繰り返し可）。
2. ツリーでシーケンスをクリック → 右側で自動プレビュー。
3. `◀ ▶` または `↑ ↓` で切替、`スペース` で再生／停止。
4. シーケンスを**ダブルクリック**または **⬇ タイムラインへ** → 再生ヘッド位置に追加、ビン自動整理。
5. **自動 Loop** をオンにするか、層を選択して **🔁 Loop** を押す。

---

## Windows フォルダー選択

- まず **Explorer 大ウィンドウ**が開きます。フォルダーへ移動し **開く** を押してください（個別 PNG の選択は不要）。
- キャンセルや失敗時は CEP ネイティブ選択器にフォールバック。
- **Shift + 「フォルダー追加」**：ステータスバーにデバッグ情報。
- 失敗時：`%TEMP%\pngseq_pick_log.txt`、`%TEMP%\pngseq_host_log.txt` を確認。
