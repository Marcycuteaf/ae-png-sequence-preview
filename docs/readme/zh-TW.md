## 功能

- 多資料夾、巢狀樹狀瀏覽、搜尋
- 選中即預覽播放，可逐格、調 FPS
- 一鍵加入時間軸（對齊播放頭）
- 自動分類專案資料夾、自訂介面配色
- Loop 按鈕與自動 Loop

---

## 介面

| 區域 | 說明 |
| --- | --- |
| 頂部 | 加入資料夾、清空、配色 |
| 左側 | 搜尋、序列樹（資料夾 / 序列） |
| 右側 | 預覽、播放控制、FPS、Loop、加入時間軸 |
| 底部 | 狀態列 |

**Windows：** Shift + 點「加入資料夾」可查看除錯資訊。

---

## 安裝

1. 下載 [Releases](https://github.com/Marcycuteaf/ae-png-sequence-preview/releases/latest) 對應平台的 `.zxp`
2. 用 [ZXP Installer](https://aescripts.com/learn/zxp-installer/) 安裝
3. 重啟 AE → **Window → Extensions → PNG 序列預覽**

---

## 操作

1. 點 **加入資料夾**，選素材根目錄
2. 左側點序列 → 右側預覽
3. 雙擊序列或按 **加入時間軸**

---

## 需求

After Effects 2019–2025+ · macOS / Windows
