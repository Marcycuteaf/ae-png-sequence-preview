## 這是什麼？

在 After Effects 裡快速瀏覽多層資料夾中的 PNG 序列、即時預覽播放，並一鍵把序列加到時間軸播放頭位置。支援多資料夾、搜尋、自動分類與介面配色。

---

## 介面說明

### 頂部工具列

| 控制項 | 說明 |
| --- | --- |
| **➕ 加入資料夾** | 選擇素材根目錄，可多次加入不同資料夾。Windows 優先開 Explorer 大視窗；**Shift + 點擊** 顯示除錯資訊 |
| **🗑** | 清空已載入的所有資料夾與序列 |
| **🎨** | 開啟／關閉介面配色面板 |
| **根目錄標籤** | 顯示目前已載入的資料夾路徑摘要 |

### 配色面板（🎨）

| 控制項 | 說明 |
| --- | --- |
| **主色 / 副色 / 背景** | 自訂面板主題色 |
| **色票預設** | 一鍵套用內建配色 |
| **重設** | 恢復預設配色 |

### 左側 — 序列樹

| 控制項 | 說明 |
| --- | --- |
| **🔍 搜尋** | 依序列名稱即時篩選 |
| **序列數量徽章** | 顯示目前找到的 PNG 序列總數 |
| **⊞ / ⊟** | 全部展開／全部收合樹狀節點 |
| **樹狀列表** | `📁` 資料夾、`🎞` PNG 序列（含張數）。點選序列即預覽 |

### 右側 — 預覽與控制

| 控制項 | 說明 |
| --- | --- |
| **預覽畫面** | 顯示目前格；未選序列時顯示提示文字 |
| **序列名稱 / 格數** | 目前序列名稱與 `目前格 / 總格數` |
| **滑桿** | 逐格拖曳預覽 |
| **◀ / ▶** | 切換上一個／下一個序列 |
| **▶ 播放** | 播放／暫停（快捷鍵：空白鍵） |
| **FPS** | 預覽播放速率 |
| **自動播放** | 切換序列時自動開始播放 |
| **自動 Loop** | 加入時間軸時自動套用 `loopOut("cycle")` |
| **🔁 Loop** | 對時間軸上**已選取圖層**套用循環 |
| **⬇ 加入時間軸** | 匯入序列並加到目前合成、對齊**播放頭**；專案面板依素材包自動分類 |

### 底部狀態列

顯示載入進度、匯入結果、錯誤或除錯訊息（Windows 選資料夾失敗時會顯示原因）。

## 安裝

1. 安裝 [aescripts ZXP Installer](https://aescripts.com/learn/zxp-installer/)。
2. 從 [Releases](https://github.com/Marcycuteaf/ae-png-sequence-preview/releases/latest) 下載對應平台的 `.zxp`。
3. 拖入安裝器完成安裝。
4. 重啟 AE → **Window ▸ Extensions ▸ PNG 序列預覽**。

**手動安裝（macOS）**：複製到 CEP 擴充目錄，並依 AE 版本開啟 `PlayerDebugMode`（2024≈CSXS.11、2025≈CSXS.12）。

---

## 基本操作

1. 點 **➕ 加入資料夾**，選含有 PNG 序列的根目錄（可重複加入）。
2. 左側樹中點選序列 → 右側自動預覽播放。
3. 用 `◀ ▶` 或鍵盤 `↑ ↓` 切換序列；`空白鍵` 播放／暫停。
4. **雙擊序列**或按 **⬇ 加入時間軸** → 加到目前合成播放頭，並在專案面板建立分類資料夾。
5. 需要循環時勾選 **自動 Loop**，或選中圖層後按 **🔁 Loop**。

---

## Windows 選資料夾

- 優先開啟 **Explorer 大視窗**：導覽到資料夾後按 **開啟**（不需選個別 PNG）。
- 若 Explorer 取消或失敗，會自動改用 CEP 原生選擇器。
- **Shift + 點「加入資料夾」**：狀態列顯示 OS、AE 版本、`system.callSystem` 是否可用。
- 仍失敗時查看：`%TEMP%\pngseq_pick_log.txt`、`%TEMP%\pngseq_host_log.txt`。
