## Что это?

Быстрый просмотр PNG-секвенций во вложенных папках прямо в After Effects, превью в реальном времени и добавление секвенции на таймлайн в позицию плейхеда. Поддержка нескольких папок, поиска, автоматических бинов и настройки цветов интерфейса.

---

## Интерфейс

### Верхняя панель

| Элемент | Описание |
| --- | --- |
| **➕ Добавить папку** | Выбор корневой папки; можно добавить несколько. В Windows сначала Explorer. **Shift + клик** — отладочная информация |
| **🗑** | Очистить все загруженные папки и секвенции |
| **🎨** | Показать / скрыть панель цветов |
| **Метка корня** | Краткая сводка загруженных путей |

### Панель цветов (🎨)

| Элемент | Описание |
| --- | --- |
| **Основной / доп. / фон** | Настройка цветов панели |
| **Пресеты** | Встроенные палитры в один клик |
| **Сброс** | Вернуть значения по умолчанию |

### Слева — дерево секвенций

| Элемент | Описание |
| --- | --- |
| **🔍 Поиск** | Фильтр по имени секвенции |
| **Счётчик** | Число найденных PNG-секвенций |
| **⊞ / ⊟** | Развернуть всё / свернуть всё |
| **Дерево** | `📁` папки, `🎞` секвенции (с числом кадров). Клик — превью |

### Справа — превью и управление

| Элемент | Описание |
| --- | --- |
| **Экран превью** | Текущий кадр; подсказка, если ничего не выбрано |
| **Имя / кадры** | Имя секвенции и `текущий / всего` |
| **Ползунок** | Покадровая перемотка |
| **◀ / ▶** | Предыдущая / следующая секвенция |
| **▶ Воспроизведение** | Play / pause (`Пробел`) |
| **FPS** | Скорость превью |
| **Автовоспроизведение** | Автозапуск при смене секвенции |
| **Авто Loop** | `loopOut("cycle")` при добавлении на таймлайн |
| **🔁 Loop** | Цикл для **выбранного слоя** на таймлайне |
| **⬇ На таймлайн** | Импорт и добавление в активную композицию на **плейхед**; автобин по имени пака |

### Строка состояния (внизу)

Прогресс загрузки, результаты импорта, ошибки и отладка (в т.ч. причины сбоя выбора папки в Windows).

## Установка

1. Установите [aescripts ZXP Installer](https://aescripts.com/learn/zxp-installer/).
2. Скачайте `.zxp` для вашей ОС из [Releases](https://github.com/Marcycuteaf/ae-png-sequence-preview/releases/latest).
3. Установите через ZXP Installer.
4. Перезапустите AE → **Window ▸ Extensions ▸ PNG 序列預覽**.

**Ручная установка (macOS)**: скопируйте в папку CEP и включите `PlayerDebugMode` для вашей версии AE (2024≈CSXS.11, 2025≈CSXS.12).

---

## Использование

1. Нажмите **➕ Добавить папку** и выберите корневую директорию (можно несколько).
2. Кликните секвенцию в дереве → превью справа с автовоспроизведением.
3. `◀ ▶` или `↑ ↓` — переключение; `Пробел` — play/pause.
4. **Двойной клик** или **⬇ На таймлайн** → добавление на плейхед с автоматическим бином.
5. Включите **Авто Loop** или выберите слой и нажмите **🔁 Loop**.

---

## Выбор папки в Windows

- Сначала открывается **большое окно Explorer**: перейдите в папку и нажмите **Открыть** (выбирать отдельные PNG не нужно).
- При отмене или ошибке используется запасной CEP-выборщик.
- **Shift + «Добавить папку»**: отладочная информация в строке состояния.
- При сбое проверьте `%TEMP%\pngseq_pick_log.txt` и `%TEMP%\pngseq_host_log.txt`.
